Atividade growtwitter com banco de dados PostgreSql integrado do curso Full Stack Web Developer.

O intuíto desta atividade era replicar as funcionalidades do Twitter usando PostgreSQL como banco de dados e Prisma como integrador

## Growtwitter

Este projeto é um clone do Twitter, construído como uma aplicação back-end usando Node.js, PostgreSQL como banco de dados e Prisma como integrador de banco de dados. Utilizando Express, controllers, services e DTOs.
O projeto é uma implementação robusta e eficiente das funcionalidades básicas do Twitter.

Lista das funcionalidades do projeto:

CRUD de usuário.

CRUD de tweet.

CRUD de replies.

likes.

follow.(seguir ou deseguir outros usuários.)

Scripts disponíveis
No diretório do projeto, você pode executar:

Instalar com:

yarn install

Executa o aplicativo no modo de desenvolvimento.
Abrir http://localhost:3333 para visualizá-lo no navegador. Rodar com:

###`yarn dev`

A página será recarregada se você fizer edições.
Você também verá quaisquer erros de lint no console.
